# OMNI Geometry MLB
 
Cross-frame angular structure analysis: a self-funded research project
investigating directed angular separations between local and celestial
reference points as predictors of binary outcomes in a 25-year MLB
sample (2000-2024, n = 57,635 events).

## Reproducibility
 
The engine (v2.0.2) is deterministic under identical inputs and
pinned package versions. Dataset integrity is enforced via SHA-256
hashing and row-count verification.
 
This repository contains both exploratory and confirmatory work.
Confirmatory analyses are defined by OSF pre-registration.
